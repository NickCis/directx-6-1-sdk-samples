Description
===========
  The Flare sample shows how to create a lens flare effect using alpha 
  blending.

Path
====
  Source: Mssdk\Samples\Multimedia\D3dim\Src\Flare

  Executable: Mssdk\Samples\Multimedia\D3dim\Bin

User's Guide
============
  Press F1 to see available commands.

Programming Notes
=================
  In this sample, lens flare is simulated with mathematical functions which 
  govern the position of each flare. The flares are rendered using additive 
  alpha blending, and animated to give a sparkle effect.
