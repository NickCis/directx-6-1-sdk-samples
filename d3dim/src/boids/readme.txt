Description
===========
  Boids illustrates how to write a Direct3D program.Using a flocking algorithm, 
  the program moves a group of 3-D objects over a simple landscape.

Path
====
  Source: Mssdk\Samples\Multimedia\D3dim\Src\Boids

  Executable: Mssdk\Samples\Multimedia\D3dim\Bin

User's Guide
============
  Press F1 to see available commands.

Programming Notes
=================
  The Boids program illustrates the fundamentals of creating a 3-D environment 
  and animating a group of objects in it. From a programming perspective, the 
  most interesting aspect is the way flocking is handled.
