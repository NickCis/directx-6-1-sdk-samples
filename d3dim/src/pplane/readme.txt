Description
===========
  The PPlane program is a simple example of how to create Direct3D Immediate 
  Mode programs.

Path
====
  Source: Mssdk\Samples\Multimedia\D3dim\Src\Pplane

  Executable: Mssdk\Samples\Multimedia\D3dim\Bin

User's Guide
============
  Press F1 to see available commands.

Programming Notes
=================
  In addition to demonstrating Direct3D Immediate Mode programming techniques, 
  this application contains a useful flocking algorithm used to animate the 
  paper planes.
