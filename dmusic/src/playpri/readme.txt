Description
===========
  The PlayPrimary sample shows how to play a primary segment based on an 
  authored music section.

Path
====
  Source: Mssdk\Samples\Multimedia\Dmusic\Src\PlayPri

  Executable: Mssdk\Samples\Multimedia\DMusic\Bin

User's Guide
============
  Click Close to quit the program.

Programming Notes
=================
  The program uses the registry key set up by the DirectX 6.0 SDK setup to
  find the media file path.

  Helper.cpp contains useful functions that set up DirectMusic. These
  functions are called from main.cpp.
