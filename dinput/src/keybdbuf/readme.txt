Description
===========
  The KeybdBuf program obtains and displays keyboard data.

Path
====
  Source: Mssdk\Samples\Multimedia\DInput\src\KeybdBuf

  Executable: Mssdk\Samples\Multimedia\DInput\bin

User's Guide
============
  Each time you press or release a key, the event is recorded on the screen. 
  Keys are identified by their index values (see Keyboard Device Constants).

Programming Notes
=================
  This sample illustrates how an application can use DirectInput to obtain 
  buffered keyboard data.
